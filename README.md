# SmartBiz OS 

SmartBiz OS is an AI-powered business dashboard designed for SMEs and startups.

##  Features

*  Revenue Dashboard
*  AI Business Assistant
*  CSV Upload & Preview
*  CSV Analysis
*  CRM Lead Management
*  Export Reports
*  API Key Integration
*  Modern Responsive UI

##  Tech Stack

* HTML5
* CSS3
* JavaScript
* Groq API
